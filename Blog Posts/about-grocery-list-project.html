About the Grocery List Project

   The Grocery List Project: How it works & looks.

   This projects aim was for practice with vectors, menu design, and outputing a useable list I could print and go shop with.  The program saves the shopping list to a text file, "groceries.txt" in the local directory of the project.

   Here is a visual break down of the program flow.

Main Menu:  Has 5 options

Add Item: Takes input of item and quantity in string format.

Delete Item: Allows deleting of single items by an item number.

Review List: Previews the shopping list without saving or printing.

Save & Print:  A one step option that creates a text file called "groceries.txt" outputs the file to the current working directory then opens it in notepad and prints it out to the default system printer.

   The program overwrites the text file each time save & print option is executed.  The shopping list can be opened in a text editor, since it is a text file saved in the local directory of the project.

   This was a very simple exercise that gave me some insights into using vectors and manipulating them.  It also presented a challenge as to the way I wanted to input my list items. 

    I wanted to be able to enter quantities but, not just 1 or 2, I wanted to be able to enter a size or container type with the item and have it combine the quantity with the item name for one record in the vector to make it easier to delete if needed.
