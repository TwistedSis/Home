IDE's (Integrated Development Environments) I've tried

My first Ide: Codeblocks

Codeblocks was a good start for the begining, but lacked a few options I wanted and needed later.

Pros:

  • Easy to use
  • Light weight
  • Decent interface
  • Low learning curve

Cons:

  • I didn't find any github integration in the version I used.
  • Commenting & Un-commenting large blocks had a wierd short cut key sequence.
  • The native compiler and linker didn't like string conversion like, stoi and atoi.


My second and current favorite IDE: Codelite

Had some issues at first but found ways to make it work better for me.

Pros:

  • Has a Github integration plugin.
  • Easy commenting & Un-commenting of large blocks with <ctrl></>
  • Customizable layouts

Cons:

  • On my windows machine the linker needed some tweaks to the path command to get to work properly. (I will post my work around for that in another post.)


There were a few other pros and cons to each IDE which I will edit into this post later.
